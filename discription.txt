# Elite Recon

Advanced reconnaissance automation framework for bug bounty and offensive security research.

## Overview

Elite Recon is a full-scale automated reconnaissance pipeline designed for bug bounty hunters and security researchers. It combines passive intelligence gathering, active discovery, endpoint extraction, deep JavaScript analysis, secret detection, and infrastructure fingerprinting into a single workflow.

The pipeline is built to automate high-volume reconnaissance while reducing noise through validation, entropy checks, and confidence scoring.

## Usage

### Basic Scan

```bash
./elite_recon.sh target.com
```

### With GitHub Token

```bash
./elite_recon.sh target.com -gt GITHUB_TOKEN
```

### Enable Active Testing

```bash
./elite_recon.sh target.com -gt GITHUB_TOKEN --active
```
